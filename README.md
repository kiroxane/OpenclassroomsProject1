Booki


Description du projet
Booki est un site web permettant aux utilisateurs de trouver des hébergements et des activités dans la ville de leur choix.

Ce projet a été réalisé dans le cadre de la formation OpenClassrooms – Développeur Web.

L'objectif du projet est d'intégrer l'interface du site à partir d'une maquette Figma en utilisant HTML et CSS, tout en respectant les bonnes pratiques du développement web.
Objectifs du projet
Créer la structure HTML du site
Mettre en forme la page avec CSS
Respecter la maquette fournie
Rendre le site responsive (adapté mobile, tablette et desktop)
Utiliser une structure de code claire et organisée

Technologies utilisées
HTML5
CSS3
Figma (maquette de référence)
Git & GitHub pour le versionnement du code
